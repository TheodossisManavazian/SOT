# SOT
Stocks and Options Tool. This tool is meant to help traders get a better understanding of the market.

## Step 1
  Make sure you're using Python 3.7.4
  In order to use this tool you need to sign up for a TD Ameritrade Developer Account and register an APP for the API key.

## Step 2
  Pip3 Install the requirements.txt file.

## Step 3 
  Change the config.py so that it reflects your API key __Make sure you keep the ending @AMER.OAUTHAP, just add your key before the '@'__
  
## Step 4
  Change the path for where your chromedriver is installed (it should be in the Externals Folder)

## Step 5
  Then run Client.py and login to your Developer account, The Token will automatically be generated after you run Client.py
  
Now when you want to run the Client, just run the SOT.py file in the Kivy folder
